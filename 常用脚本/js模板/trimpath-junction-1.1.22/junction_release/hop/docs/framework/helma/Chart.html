helma.Chart

Object
   |
   +--helma.Chart


Instances of this class are capable of reading Excel spreadsheets and rendering them as XHTML table. Internally helma.Chart uses the Java Excel API by Andy Khan.
Defined in Chart.js

Author: Tobi Schaefer
